MeteorEdit Cookie Bridge

Configuration de la connexion à votre instance

Connexion

Renseignez l'URL de votre instance MeteorEdit (HTTPS recommandé) et un token API généré depuis Profile > Tokens API sur le site.

Acceptez l'avertissement HTTPS auto-signé dans votre navigateur si nécessaire.

Le token est conservé localement dans votre navigateur. Il peut être révoqué à tout moment depuis le site.

Comment ça marche ?

  1. Connectez-vous à une plateforme supportée (Facebook, Instagram, LinkedIn, X…).
  2. Cliquez sur l'icône MeteorEdit dans la barre d'extensions.
  3. Sélectionnez le dossier de destination et confirmez l'import.
  4. Les cookies sont chiffrés localement avec la clé publique du dossier avant envoi.
  5. Seul vous, avec votre mot de passe E2E, pouvez les déchiffrer côté MeteorEdit.
